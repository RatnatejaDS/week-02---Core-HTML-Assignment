HTML Tags

  1. <html>: The root element that wraps the entire HTML document.
  2. <head>: Contains metadata about the document, such as the title, character encoding, and linked stylesheets.
  3. <title>: Sets the title of the web page, which appears in the browser's title bar or tab.
  4. <meta>: Provides metadata about the HTML document, such as character encoding and authorship.
  5. <link>: Used to link external resources like stylesheets (CSS) to the HTML document.
  6. <style>: Contains inline CSS or references an external CSS file to define the page's style.
  7. <body>: Contains the visible content of the web page, such as text, images, and interactive elements.
  8. <header>: Represents a container for introductory content or a set of navigation links.
  9. <nav>: Defines a section of navigation links, typically used within the <header> or <footer>.
  10. <main>: Represents the main content of the web page.
  11. <article>: Represents a self-contained piece of content, such as a news article or blog post.
  12. <section>: Groups related content together and provides a semantic structure.
  13. <aside>: Contains content that is tangentially related to the main content.
  14. <footer>: Contains information about the document, authorship, copyright, and contact details.
  15. <a>: Creates hyperlinks to other web pages, files, or resources.
  16. <img>: Embeds images in the document.
  17. <p>: Defines a paragraph of text.
  18. <ul>: Creates an unordered (bulleted) list.
  19. <ol>: Creates an ordered (numbered) list.
  20. <li>: Represents a list item within <ul> or <ol> elements.
