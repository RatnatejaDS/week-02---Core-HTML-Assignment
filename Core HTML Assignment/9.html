Common HTML Text Formatting Tags

  1. <strong>:
    • Use for important or strongly emphasized text.
    • Don't use for text that should be bold for styling purposes only.
  2. <em>:
    • Use for emphasizing text, such as for book titles or stress in a sentence.
    • Don't use for text that should be italicized for styling purposes only.
  3. <u>:
    • Use for underlining text to indicate links or other special content.
    • Don't use for general text underlining; use CSS for that purpose.
  4. <s>:
    • Use for strikethrough text to indicate deleted or no longer relevant content.
    • Don't use for general text decoration; use CSS for styling effects.
  5. <sup>:
    • Use for superscript text, like mathematical exponents (e.g., x2).
    • Don't use for general text; use CSS for styling effects.
  6. <sub>:
    • Use for subscript text, such as chemical formulas (e.g., H2O).
    • Don't use for general text; use CSS for styling effects.
  7. <mark>:
    • Use to highlight or mark text to draw attention to specific content.
    • Don't use for general text coloring; use CSS for styling effects.
